# Dashboard

## Installation

**1) Startup**

Clone project repository to your local directory:

```bash
git clone git@github.com:ergonode/frontend.git
```

Install project dependencies:

```bash
npm install
```

**2) Build**

Run development mode

```bash
npm run start
```

Run production mode

```bash
npm run start:prod
```
Build development mode

```bash
npm run build
```
Build production mode

```bash
npm run build:prod
```
